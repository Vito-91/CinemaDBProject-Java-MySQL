# CinemaDBProject-Java-MySQL
Applicazione Java console che utilizza JDBC per collegarsi a un database MySQL.
Il sistema permette di:

• VISUALIZZARE FILM
• VISUALIZZARE SPETTACOLI
• GESTIRE CLIENTI (inserimento e visualizzazione)
• PRENOTARE POSTI CON CONFERMA
• CONTROLLO DISPONIBILITÀ POSTI

Progetto didattico realizzato per consolidare le basi di JAVA BACKEND, SQL e GESTIONE DATABASE.
